Nortantis

Nortantis is a fantasy map generator. It uses a simple tectonic plate simulation to create islands and continents with trees, rivers, and mountains. The result has the appearance of an old-fashioned hand-drawn map.

Download and Run

Prerequisites

To run Nortantis you must have Java 7 or latter installed. If you don't already have it installed, you can find it here.

Download

Get the latest version of Nortantis here.

Run

Nortantis is run like any other executable jar. The way to do this depends on what OS you have:

  • Windows - Unzip the downloaded file, open it, and double click nortantis_map_generator.jar.
  • Linux - Unzip the downloaded file, cd to the folder with the extracted contents, and run

    • java -jar nortantis_map_generator.jar
